## Notes for self

Creating venv
``` shell
$ python -m venv <venv_name>
```

Activating venv
``` shell
$ source <venv>/bin/activate
```

Deactivating venv
``` shell
(venv_name) $ deactivate
```

Exporting enviroment variable to run flask app
``` shell
$ export FLASK_APP=file.py
```
